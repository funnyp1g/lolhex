## 1. 设计系统 (Design System)

### 1.1 色彩体系 (Color Palette)

| 名称 | 色值 | 用途 | 使用场景 |

| **Primary BG** | `#e6f7ff` | 主色背景 | 选中卡片底色、标签背景 |

#### 1.1.3 文字色

| 名称 | 色值 | 用途 |

#### 1.1.5 Tier 评级色

> 对应 PRD §6.1.4 champion_augments 集合 `tier` 字段

| Tier | 色值 | 背景色 | 含义 |
|------|------|--------|------|
| **S** | `#FF4D4F` | `#fff1f0` | 最强（胜率 TOP 5%） |
| **A** | `#FA8C16` | `#fff7e6` | 很强（胜率 TOP 15%） |
| **B** | `#FADB14` | `#fffbe6` | 尚可（胜率 TOP 40%） |
| **C** | `#52C41A` | `#f6ffed` | 一般（胜率 TOP 70%） |

#### 1.1.6 语义色

| 名称 | 色值 | 用途 |
|------|------|------|
| **Success** | `#52C41A` | 胜率上升、正向数据 |
| **Warning** | `#FAAD14` | 警告提示 |
| **Error** | `#FF4D4F` | 错误状态、网络失败 |
| **Info** | `#1890FF` | 信息提示 |

#### 1.1.7 胜率条渐变

胜率条 (rate-bar) 的颜色根据数值动态变化：

| 胜率区间 | 颜色 | 说明 |
|----------|------|------|
| ≥ 55% | `#FF4D4F` → `#FF7875` | 高胜率（红色系） |
| 50%-55% | `#FA8C16` → `#FFC069` | 中胜率（橙色系） |
| 45%-50% | `#FADB14` → `#FFF566` | 一般（黄色系） |
| < 45% | `#8C8C8C` → `#BFBFBF` | 低胜率（灰色系） |

### 1.2 字体排版 (Typography)

> 字体栈：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Helvetica Neue", "Microsoft YaHei", sans-serif`

| 级别 | 字号 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| **Display** | 24px (1.5rem) | 32px | 700 (Bold) | 英雄详情页顶部大标题 |
| **Title Large** | 20px (1.25rem) | 28px | 600 (Semibold) | 页面主标题 |
| **Title** | 17px (1.0625rem) | 24px | 600 (Semibold) | 区块标题（推荐海克斯、推荐装备） |
| **Body Large** | 15px (0.9375rem) | 22px | 400 (Regular) | 正文内容 |
| **Body** | 14px (0.875rem) | 20px | 400 (Regular) | 默认文字、卡片内容 |
| **Caption** | 12px (0.75rem) | 18px | 400 (Regular) | 辅助信息、胜率/选取率数值 |
| **Caption Small** | 10px (0.625rem) | 16px | 400 (Regular) | 版本信息、脚注 |
| **Badge** | 10px (0.625rem) | 14px | 600 (Semibold) | Tier 徽章文字 |

**文字排版规则：**
- 英雄名称使用 Title Large + Bold
- 胜率/选取率数字使用 Caption + Medium (500)
- 数值百分比统一保留 1 位小数（如 52.3%）
- 样本量数字使用千分位分隔符（如 5,286 场）
- 标题与正文之间间距 = 标题行高

### 1.3 间距系统 (Spacing)

> 基于 **4px** 网格系统，所有间距值为 4 的倍数

| Token | 值 | 用途 |
|-------|----|------|
| `--space-xxs` | 2px | 图标与文字间距、徽章内间距 |
| `--space-xs` | 4px | 紧凑元素间距 |
| `--space-sm` | 8px | 卡片内 padding、列表项间距 |
| `--space-md` | 12px | 表单元素间距、区块内间距 |
| `--space-base` | 16px | 卡片 padding、区块间距 |
| `--space-lg` | 20px | Section 间距 |
| `--space-xl` | 24px | 区块分隔间距 |
| `--space-xxl` | 32px | 页面顶部/底部留白 |

**布局常量：**

| 属性 | 值 | 说明 |
|------|----|------|
| 页面左右边距 | 16px | 所有页面统一 |
| 卡片内 padding | 12px 16px | 上下 12px，左右 16px |
| 卡片间距 | 12px | 卡片之间的垂直/水平间距 |
| 列表项高度 | 56px ~ 72px | 根据内容自适应 |
| Section 标题间距 | 20px top / 12px bottom | 区块标题上下间距 |
| 底部安全区 | env(safe-area-inset-bottom) | 适配 iPhone X+ |

### 1.7 动效规范 (Motion)

| 场景 | 时长 | 缓动函数 | 说明 |
|------|------|----------|------|
| 页面切换 | 300ms | `ease-in-out` | 左右滑动切换 |
| 弹窗出现 | 250ms | `ease-out` | 从底部滑入 |
| 弹窗消失 | 200ms | `ease-in` | 向下滑出 |
| 卡片点击反馈 | 100ms | `ease` | 缩放至 0.97 |
| 数据加载淡入 | 200ms | `ease-out` | Skeleton → Content 过渡 |
| Tab 切换 | 200ms | `ease` | 内容淡入 |

### 5.2 数据加载状态

**加载流程：**

```
页面打开
  │
  ├─ 有本地缓存？ ─── 是 → 立即显示缓存数据 (标记 "缓存数据")
  │                            │
  │                            └──→ 后台请求新数据 → 刷新页面 (无感知)
  │
  └─ 无本地缓存？ ─── 否 → 显示 Skeleton 骨架屏
                               │
                               ├─ 请求成功 → 淡入真实数据 (200ms 过渡)
                               │              └→ 写入本地缓存
                               │
                               └─ 请求失败 → 显示错误状态
                                             ├─ "网络异常，点击重试"
                                             └─ [重试] 按钮
```

**骨架屏 → 内容过渡规则：**

| 场景 | 骨架屏持续时间 | 过渡方式 |
|------|--------------|----------|
| 英雄列表 | 数据到达即切换 | opacity 0→1, 200ms |

| 海克斯列表 | 数据到达即切换 | opacity 0→1, 200ms |

```javascript
        color: lineColor,
        borderColor: '#ffffff',
        borderWidth: 2,
      },
      areaStyle: {
        color: {
          type: 'linear',
          x: 0, y: 0, x2: 0, y2: 1,
          colorStops: [
            { offset: 0, color: hexToRgba(lineColor, 0.15) },
            { offset: 1, color: hexToRgba(lineColor, 0.02) },
          ],
        },
      },
    }],

    // 动画
    animation: true,
    animationDuration: 800,
    animationEasing: 'cubicOut',
  };
}

// 工具函数: hex 转 rgba
function hexToRgba(hex, alpha) {
  const r = parseInt(hex.slice(1, 3), 16);
  const g = parseInt(hex.slice(3, 5), 16);
  const b = parseInt(hex.slice(5, 7), 16);
  return `rgba(${r},${g},${b},${alpha})`;
}

module.exports = { getLineOption };
```

**JS 逻辑：**

```javascript
// components/version-trend-chart/index.js
import { getLineOption } from './chart-option';

Component({
  properties: {
    chartData: {
      type: Array,
      value: [],
      observer: 'onDataChange',
    },
    title: { type: String, value: '' },
    lineColor: { type: String, value: '#1890FF' },
    height: { type: Number, value: 200 },
    showTooltip: { type: Boolean, value: true },
  },

  data: {
    ec: {
      lazyLoad: true,  // 延迟加载，等数据就绪
    },
    loading: true,
    error: false,
  },

  methods: {
    onDataChange(newVal) {
      if (newVal && newVal.length > 0) {
        this.setData({ loading: false });
        this.initChart();
      } else {
        this.setData({ loading: false });
      }
    },

    initChart() {
      this.ecComponent = this.selectComponent('#ec-canvas');
      const option = getLineOption(this.data.chartData, {
        lineColor: this.data.lineColor,
        title: this.data.title,
      });
      this.ecComponent.init((canvas, width, height, dpr) => {
        const chart = echarts.init(canvas, null, {
          width, height, devicePixelRatio: dpr,
        });
        chart.setOption(option);
        this._chart = chart;

        // 绑定触摸事件 (小程序 Canvas 交互)
        canvas.addEventListener('touchstart', (e) => {
          if (e.touches.length > 0) {
            var touch = e.touches[0];
            var handler = this._chart.getZr().handler;
            handler.dispatch('mousedown', {
              zrX: touch.x, zrY: touch.y,
            });
            handler.dispatch('mousemove', {
              zrX: touch.x, zrY: touch.y,
            });
            handler.dispatch('mouseup', {
              zrX: touch.x, zrY: touch.y,
            });
          }
        }, false);

        return chart;
      });
    },

    onRetry() {
      this.setData({ loading: true, error: false });
      this.triggerEvent('retry');
    },
  },
});
```
